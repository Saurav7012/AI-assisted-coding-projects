# Tic Tac Toe — React + Vite + Tailwind

A small Tic-Tac-Toe game built with React and styled using Tailwind CSS. Uses Vite as the dev server.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

Notes

- Tailwind is configured in `tailwind.config.cjs` and `postcss.config.cjs`.
- The app entry is `src/main.jsx` and components live in `src/components/`.
